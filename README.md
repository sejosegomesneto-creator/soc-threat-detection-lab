# SOC Threat Detection Lab

Laboratório de detecção de ameaças focado em Security Operations Center 
(SOC).

Este projeto simula cenários de comportamento suspeito em endpoints para 
estudo de detecção e análise de eventos de segurança.

## Cenários simulados

- Execução suspeita de PowerShell
- Processos anômalos
- Tentativa de download remoto

## Objetivo

Desenvolver habilidades utilizadas por analistas SOC para detecção de 
ameaças e investigação de incidentes.#
